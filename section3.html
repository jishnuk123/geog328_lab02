My Projects

Here are some of the main projects I’ve been developing to strengthen my technical and creative skills:

I designed and developed my own personal portfolio website using React.js to showcase my resume, projects, and technical skills. The goal of this project was to create a clean, responsive, and interactive way for others to learn more about me as a developer and data enthusiast. The website includes a dynamic project gallery, smooth navigation, and an about section that highlights my interests in AI and data engineering.

This ongoing project involves building a machine learning model that analyzes player performance statistics to optimize fantasy football lineups. I have been experimenting with regression models and predictive analytics to find patterns in player stats and matchups. The project combines my interest in sports and data science helping me apply real-world machine learning concepts in a fun and competitive context but it is still a work in progress.

Another project I have been planning is a Spotify music recommender that uses machine learning to suggest songs based on a user listening history and mood. I want the system to utilizes track features such as genre, preference, and likings while using Spotify’s API to train a recommendation model. This project has not been started yet but I think it will help explore recommendation algorithms, user-based vs. item-based filtering, and the integration of AI models with real-world APIs.
